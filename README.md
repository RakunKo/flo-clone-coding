### 🔩 기능

- 👋 회원가입 및 로그인 기능
    - 카카오톡 인증 API를 이용한 회원가입, 로그인
    
    [Kakao Developers](https://developers.kakao.com/)
    
- 💁 바텀 네비게이션바
- 🎵 앨범과 앨범 수록곡 확인
- 🎧 노래 재생과 이전/다음곡 재생
- 🔠 데이터베이스를 활용한 회원 관리 및 UI 설정
- 🔍 Open API Last.fm을 이용한 노래 검색
    
    [Last.fm | Play music, find songs, and discover artists](https://www.last.fm/home)
    
- 💓 좋아요한 앨범은 보관함에 저장
